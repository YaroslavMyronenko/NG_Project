Project for the New Generation course
The essence of the project is to develop a "Music Player" application using QT Quick and SQL
Project description:

Write a music player in QML. Important clarification - the player must be able to rewind, support playlists, support "scanning" and "searching" for music on the computer, must support working with volume, must be able to download music from a link from the network + if possible, implement streaming listening (like Internet radio).

Написать музыкальный плеер на QML. Важное уточнение - плеер должен уметь перемотку, поддерживать плейлисты, поддерживать "сканирование" и "поиск" музыки на компьютере, должен поддерживать работу с громкостью, должен уметь скачивать музыку по ссылке из сети + по возможности реализовать стриминговую прослушку (по типу интернет-радио).
